Init

代码逻辑

  • 如果存在环境变量 GOMAXPROCS 和 MY_CPU_LIMIT 设置 runtime.GOMAXPROCS
    runtime.GOMAXPROCS 为 go 程序所能占用的最大核数，也可以理解为线程数
  • 读取配置，配置优先级
    配置文件 > 环境变量 > 参数
  • 初始化 init Data bus config
  • 初始化 access logger
  • 初始化 kitc 的 call logger
  • 创建 RPCServer 的实例
