保护接口

Net 类的保护接口如下

保护接口主要是关于以下几方面

  • 根据给定 name 查找 blob
  • 根据给定 name 查找 layer
  • 根据给定 type 定制 layer
  • 根据给定 type 创建 layer
  • 根据给定 index 创建 layer
  • 根据 layer_index 指定的层向前传播计算
