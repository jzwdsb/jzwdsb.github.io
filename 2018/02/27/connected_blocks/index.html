连通块问题

发表于 2018-02-27 更新于 2022-10-18 分类于 algorithm
DFS, BFS

　　连通块问题的考察形式非常多，直观的比如区域中有多少个连通块，最大的连通块的大小，不太直观的形式有符号识别，利用不同符号的中洞的个数不同识别，也是求连通块的问题，例如 UVa 1103.

　　连通块实现的方式主要有 DFS 和 BFS ．

　　DFS 和 BFS 的实现都要对每个点打上标记表示是否已经经过搜索．
　　DFS 的实现利用递归会很简单.
